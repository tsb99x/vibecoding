# Text Adventure — Space Station

A click-choice text adventure game set on a damaged space station. You wake from cryosleep as the only surviving crew member and must explore the station, gather items, solve puzzles, and determine your fate.

## Features

- **12 interconnected rooms** — Cryo Bay, Central Corridor, Command Bridge, Engineering, Airlock, Storage, Med Bay, Observation Deck, Server Room, Cargo Bay, Living Quarters, Reactor Room
- **Inventory system** — Collect items like Medkits, Flashlight, Wrench, Oxygen Tank, Override Code, Critical Component, and Crew Log
- **Conditional choices** — Certain actions and paths are only available with the right items or knowledge
- **Health tracking** — Radiation and hazards can damage you; Medkits can restore health
- **Multiple endings** — Repair the station, escape pod, or perish
- **Terminal aesthetic** — Retro terminal-inspired UI with typewriter-style text animation
- **Catppuccin Mocha theme** — Dark-only theme with a carefully crafted color palette

## How to Play

Open `index.html` in a web browser and click "Begin Adventure" to start. Navigate by clicking choices presented at the bottom of the terminal. No typing required.

# AGENTS.md — Flash Cards

## Project Overview
Chinese HSK vocabulary flashcards with flip card UI: Chinese characters on front, pinyin + English translation on back.

## Flashcard Features
- Navigation: Previous/Next buttons, keyboard (arrow keys)
- Shuffle mode to randomize order
- Progress tracking (e.g., "5 / 150")
- Search/filter functionality

## Card UI
- Large Chinese characters centered on front
- Pinyin and translation on back
- Search bar for filtering
- "Back to Main Page" button (`../index.html`)

## Sizing
Use relative units (`rem`, `em`, `%`, `vh`/`vw`) — no hardcoded `px` values.

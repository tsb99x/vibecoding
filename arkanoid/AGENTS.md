# AGENTS.md — Arkanoid

## Project Overview
Classic Arkanoid (brick breaker) SPA. Player controls paddle to bounce ball and destroy bricks. Game ends when ball falls or all bricks destroyed.

## Game Mechanics
- **Canvas:** 480x600 pixels
- **Paddle:** 80px wide, 12px high, horizontal movement
- **Ball:** 8px radius
- **Bricks:** 60px wide, 18px high, 5 rows × 7 columns
- **Initial Speed:** 4 px/frame
- **Lives:** 3
- **Controls:** Mouse or arrow keys / A,D

## UI Elements
- Start/Pause button, Score and lives display
- Game over / win overlay with restart button
- "Back to Main Page" button (`../index.html`)

# Vibecoding

A repository for vibecoding experiments. This project explores the actual capabilities of vibecoding, pushes the boundaries of this approach, and investigates how AI technology can be integrated into my development workflow.

## Tech Stack

- **Backend**: [Ollama](https://ollama.ai/)
- **Model**: [Qwen3.6-27b](https://ollama.com/library/qwen3.6:27b)
- **Plugin**: [Cline](https://github.com/cline/cline) for [VSCodium](https://github.com/VSCodium/vscodium)

## Documentation

- [How to Run Ollama in Podman](https://tsb99x.github.io/notes/ollama-in-podman.html) - My own article on running Ollama within a Podman container, published on my blog.

## Workflow

This repository follows a hands-off file editing approach:

- **File Editing**: All files are edited exclusively by the AI model via the Cline plugin. This includes this README.md. I do not manually edit any files.
- **Git Management**: I manually manage the Git repository (commits, pushes, branches, etc.).
- **Local Environment**: Everything runs locally and self-hosted. No cloud services or external APIs are used.

# AGENTS.md — Linux TLDR Commands Page

## Project Overview
Quick reference for essential Linux commands, organized by category with search and copy-to-clipboard.

## UI / UX
- Commands grouped by category (File Operations, Permissions, System Information, etc.)
- Each card shows name, description, and example(s) with copy button
- Examples have visual spacing (`margin-bottom` on `.command-example`, reset with `:last-child`)
- Search filters by name, description, category, and example text
- Responsive at 600px breakpoint

## Content
- TLDR-style: short, practical examples with inline comments (`# comment`)
- Copy button copies only command part (before `#`), not comment

# Linux TLDR Commands

A simple single-page web application serving as a quick reference for essential Linux commands, organized by category with search functionality and copy-to-clipboard support.

## Features

- **Category-Based Organization** — Commands grouped by File Operations, Permissions, System Information, Text Processing, Network, Archives, Package Management, and Process Management
- **Search Functionality** — Filter commands by name, description, category, or example text
- **Copy to Clipboard** — One-click copy for any command example (copies only the command, not inline comments)
- **TLDR-Style Content** — Short, practical examples with inline comments explaining usage
- **Responsive Design** — Mobile-friendly layout with adaptive card design

## Categories Covered

| Category | Commands |
|----------|----------|
| File Operations | ls, cd, cp, mv, rm, mkdir, touch, cat, less, find |
| Permissions | chmod, chown |
| System Information | ps, top, df, free, uname, whoami |
| Text Processing | grep, sed, awk, cut |
| Network | curl, wget, ping, ssh, ip |
| Archives | tar, zip, gzip |
| Package Management | apt, yum |
| Process Management | kill, nohup |
